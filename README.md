# Pokemon Machine Learning Analysis

This project applies machine learning techniques to predict Pokemon primary types based on stats and physical characteristics.

## Models Used
- Decision Tree
- Random Forest
- KNN
- Neural Network
- PCA

## Tools
- R
- R Markdown
- ggplot2
- caret
- randomForest

## Dataset
Pokemon dataset containing stats, height, weight, abilities, and types.

## Results
Random Forest achieved the highest classification accuracy among the tested models.
